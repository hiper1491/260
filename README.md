# 13月亮曆的小共時

Dreamspell 卓爾金曆共時訊息抽籤遊戲

## 功能特色

- 抽取 KIN 1-260 的共時訊息
- 每個 KIN 顯示對應的圖騰名稱與共時訊息
- 紅、白、藍、黃四色區分不同 KIN
- 抽籤動畫與音效效果
- 抽取紀錄回顧功能

## 使用方式

1. 開啟 `index.html`
2. 點擊「抽取共時」按鈕
3. 查看抽到的 KIN 與共時訊息

## 技術

- 純 HTML / CSS / JavaScript
- 無需安裝任何套件
